The open roadbook format (v1)

A small, sim-agnostic JSON format for a rally-raid roadbook: an ordered list of tulip boxes — each a junction diagram, a cumulative distance, a compass heading (CAP) and optional pictograms. It is the contract every tool here reads and writes (the in-game plugin, the web viewer + builder, the mobile reader, the printable sheets), and the keystone that lets other sims, apps and a marketplace interoperate.     Field reference

    PathTypeMeaning
    schemaVersionint = 1Format version (required).
    trackName / trackIdstringHuman + machine track id.
    metaobjectProvenance (generatedBy, sourceRide) + world calibration (worldUpAxis, capNorthOffsetDeg).
    boxes[]arrayThe roadbook, one tulip box per instruction, in travel order (required, ≥1).
    boxes[].distTotalKmnumberCumulative distance from start; non-decreasing (required).
    boxes[].distPartialMnumberMetres since the previous box.
    boxes[].capDegint 0–359Compass heading at the box, 0=N clockwise (required).
    boxes[].typeenumstart straight turn hairpin finish (required).
    boxes[].turnDirenumleft right none.
    boxes[].dangerLevelint 0–3Severity (! .. !!!).
    boxes[].pictograms[]string[]FIA/Dakar sign codes (caution, dune, water…).
    boxes[].tulip.points[][x,y][]Junction polyline, travel-up, each in 0..1; entry first (≥2, required).
    boxes[].crossDeg / branchDeg[]number(s)Crossing road / un-taken branch bearings.
    route[][x,z,distM,elev][]Optional source polyline.

  "schemaVersion": 1,
  "trackName": "Demo stage",
  "meta": { "generatedBy": "builder", "totalDistanceKm": 0.31 },

    { "index": 1, "distTotalKm": 0.31, "distPartialM": 310, "capDeg": 78,
      "type": "turn", "turnDir": "right", "dangerLevel": 1, "pictograms": ["caution"],

  ]
