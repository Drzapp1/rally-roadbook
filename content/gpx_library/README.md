# GPX rally stage library

**Original practice stages** (not real-event data), generated as GPX tracks and
run through the plugin's GPX import pipeline. Each ships as a ready-to-study
roadbook (`roadbook_stage_*.json`) plus its source `.gpx`.

| Stage | Terrain |
|-------|---------|
| Desert Loop | sweeping pistes, dune undulation, crests |
| Mountain Pass | climbing switchbacks (hairpins) |
| Coastal Sprint | fast, gentle curves |
| Canyon Run | twisty with dips / water crossings |

## Use in-game
Copy a `roadbook_stage_*.json` into
`Documents\PiBoSo\MX Bikes\Roadbook\import\`, then **F4 → Roadbooks → import**.

## Import your own real GPX
```
python tools/rbtool.py gpx your-route.gpx "Stage name"
```
Any GPX track (lat/lon + elevation) becomes a roadbook — tulips, distances,
headings and terrain signs are generated by the same engine. Regenerate this
library with `python tools/make_gpx_library.py`.
